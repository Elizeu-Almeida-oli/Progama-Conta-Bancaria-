# 💳 Sistema Bancário em Python

Este projeto foi desenvolvido durante meus estudos de **Python**, com o objetivo de praticar lógica de programação e criação de sistemas interativos no terminal.

O programa simula um **sistema bancário simples**, permitindo realizar operações básicas como depósito, saque e visualização de extrato.

---

## 📌 Funcionalidades

O sistema possui um menu interativo com as seguintes opções:

1. Depositar dinheiro
2. Sacar dinheiro
3. Visualizar extrato
4. Encerrar o programa

---

## 🧠 Conceitos de Python utilizados

Durante o desenvolvimento foram utilizados diversos conceitos da linguagem Python, como:

- Criação e uso de **funções**
- Estruturas condicionais (`if` e `else`)
- **Laço de repetição `while`** para manter o sistema em execução
- Manipulação de **listas** para armazenar o extrato das operações
- Uso de **variáveis globais** para controle do saldo

---

## ⚙️ Estrutura do programa

O código foi dividido em funções para organizar melhor o sistema:

- `mostrar_menu()` → Exibe o menu principal
- `depositar()` → Realiza depósitos na conta
- `sacar()` → Realiza saques do saldo disponível
- `ver_extrato()` → Mostra o histórico de operações realizadas

O sistema utiliza uma **lista chamada `extrato`** para registrar todas as movimentações realizadas na conta.
